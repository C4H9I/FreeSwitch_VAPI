Это проект голосового ассистента, который работает совместно с freeswitch 

**Описание**
Freeswitch подключается по SIP к провайдеру с номером 78692470006
все входящие звонки на этот номер он перенаправляет на голосового агента, который является скриптом на python 3.10 (текущая версия в разработке)
Код голосового агента находится в папке ./voice-bot
Файл запуска server.py
при запуске файла бот начинает слушать на 127.0.0.1:8084 и с помощью ESL управляет звонком.
Задача бота поднять трубку, выдать приветствие заданное промтом или настройками, выслушать звонящего, получить его речь, трансформировать в текс с помощью YandexSpeachKit получить текст и отправить языковой модели.

Скрипт писался разными разработчиками, файл server.py точно проверен и умеет 
Файлы в voice-bot/app тоже проверены с помощью тестов ./test_api.py ./test_real_llm.py ./test_yandex.py ./test_hybrid.py



**Данные**
переменные, токены и т.д. находятся в файле .env
